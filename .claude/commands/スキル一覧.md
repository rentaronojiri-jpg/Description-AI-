# スキル一覧

このプロジェクト（AI事務）で使えるスラッシュコマンド（カスタムスキル）の一覧。

> **使い方**: チャット欄に `/スキル名` と入力して実行する。

---

## カスタムスキル（プロジェクト固有）

| コマンド | 概要 | 主な用途 |
|----------|------|----------|
| `/見積書作成` | 現場写真から工事内容を読み取り、見積書HTMLを自動生成 | 新しい見積書を作りたいとき |
| `/業者一覧作成` | `業者見積もり/` のPDFをすべて読み込み、業者連絡先一覧表（HTML）を生成・更新 | 業者が増えたとき・一覧を最新化したいとき |
| `/エクセル修正スキル` | openpyxl で生成した Excel の罫線・文字切れを修正 | 見積書テンプレートの罫線が途切れる・文字が切れるときに実行 |
| `/スキル一覧` | このファイルを表示する | 使えるスキルを確認したいとき |

---

## システム標準スキル（Claude Code 組み込み）

| コマンド | 概要 |
|----------|------|
| `/code-review` | コードのバグ・改善点をレビュー（`ultra` オプションで深い多段階レビュー） |
| `/verify` | 変更が正しく動作するかアプリを実際に動かして確認 |
| `/simplify` | コードの整理・簡略化（バグ修正ではなく品質向上） |
| `/security-review` | セキュリティ観点でブランチの変更をレビュー |
| `/run` | プロジェクトのアプリを起動して動作確認 |
| `/init` | CLAUDE.md（コードベースのドキュメント）を初期生成 |
| `/schedule` | 定期実行するクラウドエージェントを作成・管理 |

---

## スキルの追加方法

`.claude/commands/` フォルダに `スキル名.md` を作成するだけで自動的にコマンドとして登録される。

```
AI事務/
└── .claude/
    └── commands/
        ├── エクセル修正スキル.md   ← /エクセル修正スキル
        └── スキル一覧.md           ← /スキル一覧
```
