# 見積書作成スキル

現場の写真・動画フレームから工事内容を読み取り、見積書HTMLを自動生成するスキル。

## 使い方

```
/見積書作成
```

実行後、Claude が以下の情報を確認する。不明な項目はデフォルト値を使用する。

---

## 実行手順

### 1. 入力情報を収集する

ユーザーに以下を確認する（まとめて1回で聞く）：

- **現場写真・画像**：チャットに貼り付けてもらう（すでにある場合はスキップ）
- **顧客名**：見積書の宛名（例: 田中 太郎 様）
- **工事場所**：住所または略称（例: 練馬区関町）
- **工事名**：任意（不明なら画像から推定）
- **会社情報**：施工会社名・住所・TEL（初回のみ。2回目以降はCLAUDE.mdに追記して省略可）

### 2. 画像から工事内容を分析する

画像を読み込み、以下を判断する：

- 工事の種類（塗装・解体・防水・内装など）
- 施工箇所（外壁・屋根・床・天井など）
- 規模感（面積・数量の目安）
- 特記事項（劣化具合・危険箇所・養生の必要性など）

### 3. 単価・数量を決定する

`AI学習用/` フォルダのPDFサンプルを参照し、実勢価格に合わせて単価を設定する：

```
AI学習用/20260514常総市ガレージお見積り.pdf
AI学習用/20260526恵比寿お見積り.pdf
AI学習用/20260601お見積り.pdf
AI学習用/練馬区関町 リノベーション工事御見積書0425.pdf
```

単価が不明な項目は「要確認」と記入し、後でユーザーが編集できるようにする。

### 4. HTMLファイルを生成する

`見積もり作成/見積書テンプレート_単価編集用.html` をベースに、以下を埋め込んだHTMLを生成する：

- 見積番号：`YYYY-MMDD-001` 形式（今日の日付）
- 見積日付・有効期限（1ヶ月後）
- 顧客名・工事名・工事場所・工期予定
- 明細行（セクション別に分類）
- 合計金額・消費税（10%）・税込総額

**出力先**：`見積もり作成/見積もり生成/output/`
**ファイル名**：`見積書_<工事種別>_YYYYMMDD.html`

### 5. 完成を報告する

生成したファイルのパスを伝え、以下を案内する：

- ブラウザで開いて金額・数量を直接編集できること
- 「印刷 / PDF保存」ボタンでPDF化できること
- 修正が必要な場合はチャットで伝えてもらえばすぐ対応できること

---

## 出力ルール

- 消費税は10%
- 金額はすべて円（¥）表記、カンマ区切り
- 数量の単位：面積→㎡、長さ→m、個数→式・箇所・枚
- セクション例：「仮設工事」「下地処理」「塗装工事」「付帯工事」「諸経費」
- 備考欄に特記事項・注意点を記載する

## デフォルト会社情報（変更があればCLAUDE.mdに追記する）

施工会社名・住所・TELが不明な場合は `○○` のままにして、ユーザーが後で編集する。
