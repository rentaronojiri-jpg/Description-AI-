# 業者一覧作成スキル

`業者見積もり/` フォルダ内のPDFをすべて読み込み、業者の連絡先一覧表（HTML）を生成・更新するスキル。

## 使い方

```
/業者一覧作成
```

---

## 実行手順

### 1. PDFを収集する

`業者見積もり/` 配下のサブフォルダを含むすべての `.pdf` ファイルを列挙する。

```powershell
Get-ChildItem -LiteralPath "業者見積もり" -Recurse -Filter "*.pdf"
```

### 2. 各PDFから業者情報を読み取る

PDFを1件ずつ読み込み、以下の項目を抽出する：

| 項目 | 備考 |
|---|---|
| 業者名 | 見積書の発行元（右上または署名欄） |
| 工種 | 工事の種類（塗装・内装・給排水など） |
| 郵便番号・住所 | 業者の所在地 |
| 電話番号 | TEL欄 |
| FAX番号 | あれば |
| 担当者名 | 担当者欄（なければ「―」） |
| 適格請求書登録番号 | T始まりの番号（なければ「―」） |

同じ業者が複数PDFに登場する場合は重複せず1行にまとめる。

### 3. HTMLを生成する

以下のフォーマットで `業者見積もり/業者一覧表.html` を上書き保存する。

**テーブル列（金額・工事内容は含めない）：**
- 業者名
- 工種
- 住所
- 電話番号
- 担当者
- 適格請求書登録番号

**デザインルール：**
- ヘッダー背景色：`#2c3e50`（ダークネイビー）
- 偶数行背景：`#f7f8fa`
- 工種はバッジ（角丸タグ）で表示
- 印刷・PDF化に対応したレイアウト（`@media print` 対応）
- 作成日を `<p class="meta">` に記載

**出力先**：`業者見積もり/業者一覧表.html`

### 4. 完成を報告する

登録業者数と出力ファイルパスを伝える。
