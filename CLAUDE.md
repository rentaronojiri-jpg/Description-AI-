# AI事務 プロジェクト

建設工事に関わる事務作業（見積もり作成・予算管理・資料作成）を AI で効率化するプロジェクト。

## プロジェクト構成

```
AI事務/
├── CLAUDE.md                  # このファイル
├── AI学習用/                   # 学習用サンプル見積書PDF
│   ├── 20260514常総市ガレージお見積り.pdf
│   ├── 20260526恵比寿お見積り.pdf
│   ├── 20260601お見積り.pdf
│   └── 練馬区関町 リノベーション工事御見積書0425.pdf
└── 見積もり作成/
    ├── claude-code-commands.md        # Claude Codeコマンド一覧
    ├── 見積書テンプレート_単価編集用.html  # 見積書HTMLテンプレート
    └── 動画見積もりツール/
        ├── input/                     # 入力ファイル（画像・動画フレーム等）
        └── output/                    # 生成した見積書（HTML・PDF）
```

## 主な業務内容

### 1. 見積もり作成
- 現場写真・動画から工事内容を読み取り、見積書を作成する
- 出力形式は HTML（印刷・PDF変換対応）
- テンプレートは `見積もり作成/見積書テンプレート_単価編集用.html`
- 完成ファイルは `見積もり作成/動画見積もりツール/output/` へ保存
- ファイル名形式: `見積書_<工事種別>_YYYYMMDD.html`

### 2. 予算管理
- 工事案件ごとの予算・実績を管理
- 今後対応予定

### 3. 資料作成
- 工事報告書・提案書などのドキュメント作成
- 今後対応予定

## 見積書作成のルール

- 単価・数量は `AI学習用/` のPDFサンプルを参照して実勢価格に合わせる
- 消費税は10%
- 金額はすべて円（¥）表記
- 見積書には会社名・工事名・作成日・有効期限を必ず記載
- HTMLはブラウザで印刷してPDF化することを前提にレイアウトする

## 注意事項

- 顧客情報・金額情報が含まれるため、外部サービスへのアップロードは行わない
- PDFサンプルは学習・参照用のみ。出力ファイルには含めない
